How To Play

Press Space to Jump. If you hold space, you go further up. If you touch the pipes or the ground, you die and you have to restart. You get a point if you successfully pass a set of pipes. Try to get the highest score that you can possibly get as there is a "Global High Score" feature in which it stores the highest score anyone on the game has gotten. So with that all out of the way, good luck!
